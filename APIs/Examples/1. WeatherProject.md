# Weather Project

# 1. Create New Directory 'WeatherProject'
```
// Git Bash
mkdir WeatherProject
```

# 2. Move into project folder
```
cd WeatherProject
```

# 3. Create New files 'app.js' & 'index.html'
```
touch app.js index.html
```

# 4. Run VSCode
```
code .
```

# 5. Initialise npm
```
npm init
```
